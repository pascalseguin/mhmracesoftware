Import Chip Assignments from Spreadsheet

Paste a CSV or tab-separated table (copied from Excel / Google Sheets) with per-activity chip columns. Activity type is detected from the column header keywords Hike, Paddle, and Bike. Team names are fuzzy-matched to existing racers. You'll see a preview before anything is saved. Missing chips are added to the library automatically.

Expected headers (order flexible, Class column is ignored):
Team Name · Class · Hike Chip Name · Hike Chip SI Code · Paddle Chip Name · Paddle Chip SI Code · Bike Chip Name · Bike Chip SI Code

Tips

  • You can add control names after import from the Courses page.
  • A control with the same SI code can appear in multiple courses with different point values — just import once per course using the course selector above.
  • Existing controls (same SI code in same course) will have their points updated, not duplicated.
  • Teams are not automatically enrolled in courses — do that from the Racers page after import.
