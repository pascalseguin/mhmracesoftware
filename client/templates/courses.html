ⓘ SI Code Numbering & Offset — How to avoid mismatches

The problem

The SportIdent library reads only 8 bits of the station's 12-bit code number. Stations numbered 256–511 come back as actual − 256 — so station 310 arrives as code 54. The SI CN Offset setting (default 256) corrects this by adding 256 back before matching punches to controls.

The clean fix — pick one range

Collisions are impossible if all your stations stay within a single range. Choose one and stick to it:

  • Stations 1–255: set SI CN Offset to 0.
  • Stations 256–511: keep SI CN Offset at 256 (default).

Mixed ranges

The software checks each punch against configured controls first. If the raw code matches it uses it as-is; otherwise it applies the offset. Collision only occurs if you configure both control 20 and control 276 in the same course.

Reserved SI codes

  • 1 Clear · 2 Check · 3 Start · 4 Finish
  • 148 Start (old) · 149 Finish (old)

Change offset in Settings → SI CN Offset.
