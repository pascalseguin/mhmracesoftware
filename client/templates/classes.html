How Classes Work

  • Use Edit → Default Courses on each class to link it to one or more courses.
  • When a class is assigned to a team, they are automatically enrolled in that class's default courses.
  • A team's class score is the sum of their points across all courses they entered.
  • The Results page shows each team's score per course and their overall rank within their class.
  • Teams with no class assigned appear in an "Unclassified" section at the bottom.
