Add New Type

The Default Points value is a positive magnitude — whether it adds or subtracts depends on the Category. You can override the amount per-application in the Adjustments page.

How to use

  1. Create your bonus/penalty types here (e.g. Photo Challenge = +5, Gear Violation = −10).
  2. Go to Adjustments — each team has an Apply Type section.
  3. Pick the type from the dropdown. The default point value is pre-filled — change it if needed for that specific team.
  4. The Leaderboard (Class Results) shows Bonuses, Penalties, and Manual Adj as separate columns.
